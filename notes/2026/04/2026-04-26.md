## Remix IDE

### 테더 컨트랙트 코드

MyUSDT.sol

```
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

// OpenZeppelin의 표준 ERC20 라이브러리를 가져옵니다.
import "@openzeppelin/contracts/token/ERC20/ERC20.sol";
import "@openzeppelin/contracts/access/Ownable.sol";

contract MyUSDT is ERC20, Ownable {
    // 생성자: 토큰 이름(Tether USD), 심볼(USDT) 설정
    // 초기 발행량을 설정합니다 (예: 1,000,000개)
    constructor(uint256 initialSupply) ERC20("Tether USD", "USDT") Ownable(msg.sender) {
        // decimals는 기본 18자리입니다.
        _mint(msg.sender, initialSupply * 10 ** decimals());
    }

    // 관리자(Owner)만 추가로 토큰을 발행할 수 있는 기능 (필요 시)
    function mint(address to, uint256 amount) public onlyOwner {
        _mint(to, amount);
    }
}
```

배포
1. Environment 설정
    - 테스트용: Remix VM (Cancun) 선택
    - 실제: Injected Provider - MetaMask
2. initialSupply 칸에 원하는 수량 입력 (예: 1,000,000)
3. transact 버튼 클릭 (메타마스크 연결 시 가스비 승인창 연결)

컨트랙트 실행 및 전송 확인
- Deployed Contracts 목록에 내 컨트랙트가 나타남
- transfer: 특정 주소로 USDT를 보낼 때 사용
    - to에 받을 주소, amount에 수량을 입력
    - balanceOf: 특정 지갑의 잔액을 확인
    - totalSupply: 현재 발행된 총략 확인
